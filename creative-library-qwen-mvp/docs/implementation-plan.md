# 创意库 Qwen 真实解析 MVP 实施计划

## 一、服务骨架

- 建立 FastAPI 应用、静态页面服务、运行目录和健康检查。
- 定义配置、任务、元数据字段、分镜和情绪曲线的数据模型。
- 为配置脱敏、文件校验和 JSON 存储编写单元测试。

## 二、模型与任务链路

- 实现百炼 OpenAI 兼容客户端，使用 Qwen3.5-Omni 分析 Base64 视频。
- 将现有 54 项视频字段固化为解析清单，要求模型逐项返回。
- 实现后台任务、进度、错误、重试和本地 JSON 持久化。
- 提供无网络测试使用的 Mock Qwen 客户端。

## 三、页面接入

- 在顶部增加模型配置入口和连接状态。
- 将模拟上传改为真实 multipart 上传和任务轮询。
- 将结果写入现有页面状态，驱动元数据、分镜和情绪曲线展示。
- 人工确认、驳回和编辑后同步服务端 JSON。

## 四、验证

- 运行后端单元及接口测试。
- 校验前端脚本语法和 54 项字段覆盖。
- 使用 Mock 模式完成端到端上传、进度、结果和回写验证。
- 启动本地服务，检查桌面页面无控制台错误和布局异常。
