# 创意库浅色改版与真实视频流程设计

## 目标

将创意库恢复为柔和浅色工作台；保证所有弹窗均可主动关闭；把 AI 识别的视频基础信息变为可编辑、可重新归属剧集的数据；阻止 Token Plan 凭证误用于应用后端视频分析。

## 页面与交互

- 全局使用暖灰背景、雾灰面板、低饱和红色操作强调、绿色确认状态，不使用大面积纯白或深色底。
- 所有弹窗统一具备右上角关闭按钮、底部取消或关闭按钮，并支持点击遮罩和 Escape 退出。
- 单视频分析头部新增“基础信息”区域，展示剧名、集号、集标题、题材、平台、时长、本集目标、状态变化和分析重点，并提供统一编辑按钮。
- 编辑视频基础信息时，剧名可选择现有剧集；选择“新增剧名”后填写新名称并创建剧集。保存后视频归属、剧集导航和分析 JSON 同步更新。
- 原有逐条元数据编辑、审核和讨论能力保持不变。

## 模型配置

- 本应用只接受百炼按量付费凭证：普通 `sk-` API Key 与 `https://dashscope.aliyuncs.com/compatible-mode/v1`。
- 检测到 `sk-sp-`、`token-plan` 或 Coding Plan 地址时，在保存/测试阶段直接返回中文说明，不再等视频上传后失败。
- 视频继续采用百炼临时 OSS 上传，再以 `oss://` 地址交给 `qwen3.5-omni-plus-2026-03-15`；上传凭证失效或鉴权失败时返回可执行的配置建议。

## 数据结构

- `VideoOverview` 增加 `series_title` 与 `episode_title`，保留 `title` 兼容现有分析文件。
- AI 输出明确区分剧名、集标题和集号；旧结果缺少新字段时由后端兼容补齐。
- 人工编辑通过现有分析更新接口写回本机 JSON；浏览器状态同步保存到 localStorage。

## 验收

- 页面在桌面和移动端无深色残留、文字可读、无大面积纯白。
- 任一弹窗均可通过右上角、底部、遮罩和 Escape 关闭。
- 视频基础信息可编辑，现有剧集选择和新增剧集均能正确更新归属。
- Token Plan 配置在上传前被拦截并说明解决方法；按量付费配置可继续临时 OSS 上传流程。
